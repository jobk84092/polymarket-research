# Validation Checklist

## Data

- [ ] Raw inputs archived in `data/raw`
- [ ] Cleaning steps reproducible in `data/interim`
- [ ] Final datasets versioned in `data/processed`

## SQL

- [ ] Source assumptions documented in `sql/00_sources.md`
- [ ] Query outputs saved in `sql/outputs`

## Reporting

- [ ] Draft saved in `reports/drafts`
- [ ] Final approved report saved in `reports/final`

## Evidence

- [ ] Supporting screenshots saved in `evidence/screenshots`
- [ ] Export artifacts saved in `evidence/exports`
- [ ] Execution logs saved in `evidence/logs`
